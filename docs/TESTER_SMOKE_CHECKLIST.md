# ASAB (عصب) — قائمة فحص للتيستر (Smoke Test Checklist)

> الهدف: تمشية سريعة ومنظّمة على كل أدوار النظام للتأكد إن الربط بالـ backend شغّال
> قبل أي اختبار تفصيلي. علّم ✅ على اللي اشتغل و❌ على اللي وقع، وسجّل التفاصيل
> في **نموذج تقرير الأخطاء** آخر الملف.

- **الـ Frontend:** React 19 + Vite — `artifacts/mockup-sandbox`
- **الـ Backend:** Laravel 12 · `https://ivory-snail-183262.hostingersite.com`
- **اللغة:** عربي أولاً (RTL) — مع زر تبديل English في كل شاشة
- **العملة:** ريال سعودي (الأرقام جاية بالـ halalas من الباك وبتتعرض SAR)
